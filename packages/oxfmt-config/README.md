# Poool Oxfmt config

> Common Oxfmt rules we share between projects at Poool

## Installation

```bash
yarn add oxfmt @poool/oxfmt-config --dev
```

## Usage

```json
{
  "$schema": "node_modules/oxfmt/configuration_schema.json",
  "extends": [
    "node_modules/@poool/oxfmt-config/lib/recommended.json"
  ]
}
```
